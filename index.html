    Welcome to Tau, An Event Correlation Platform

    Tau is a Rule based Real-time Event Correlation Platform, it's fast and scalable and will provide you basic capabilities to work with streams of log and metric data points with minimal programming.

    Tau is designed as a real-time platform from the ground up and performance is it's foundation stone.

    How to use deploy it?

    Easiest way to deploy Tau is using the Docker compose file, just download it and run docker-componse up.
